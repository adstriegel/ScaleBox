Welcome

Welcome to the Fast Mobile Network Characterization (FMNC) web tool! This particular tool is a web instantiation of the FMNC tool suite. The tool is part of a larger project instrumenting wireless performance by Prof. Aaron Striegel at the Wireless Institute at the University of Notre Dame.

How it works

Welcome to the Fast Mobile Network Characterization (FMNC) web tool! This particular tool is a web instantiation of the FMNC tool suite. The tool suite allows you to test how well your particular network connection is performing. The tool is part of a larger project instrumenting wireless performance by Prof. Aaron Striegel at the Wireless Institute at the University of Notre Dame.

Notes

Want more information about the FMNC project? Visit our Wiki which contains details with regards to our current research based on data coming from this tool. Are you mobile? Come grab the Android app or iOS app from the respective App Stores. The tool is free to use and offers the ability to see the quality of your WiFi or cellular connection over time. Support for FMNC is provided by the industrial partners of the Wireless Institute as part of the BWAC IU/CRC site. The Notre Dame site for BWAC is supported in part through the National Science Foundation (NSF) through Grant 14-XXXXX. Further development support has been provided as well through NSF Grant 09-XXXXX and collaborations with Sprint and Alcatel-Lucent.

Privacy

The research for this project is considered IRB EXEMPT as classified by the Notre Dame Institutional Review board (pending). All core identifying information (source IP, source port) are hashed for proper anonymization before committal to the FMNC database. If you wish to have this particular connection purged from the database, please click here. Note that the purge link has a roughly 5 minute expiration.
